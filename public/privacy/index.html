Privacy

What we do

  • Completely anonymous. No account required. We don’t ask for your name, email, or phone.
  • Guest sessions only. A random session id lives in your browser so the game works. It isn’t tied to you.
  • Only de‑identified gameplay snapshots are saved. They power async multiplayer and research.
  • Your runs can help science. We share anonymized data publicly so others can learn from it.
  • Opt‑out anytime. You can opt‑out of research exports and keep playing normally.

What we do not collect

  • No names, emails, phone numbers, or login credentials.
  • No precise timestamps tied to you, no user agent strings in the dataset, no IPs in public data.

Light technical notes

  • Requests include a random session id stored locally. The server issues a short‑lived token to prevent abuse.
  • Saved snapshots are stripped of personal identifiers and contain only the decisions needed to reconstruct a match.
  • Public exports use rounded times and compact fields. See the API page for schema and license.
